
# 2020 Ray4.0 專案 - 臺北市市民服務大平台


### 臺北市市民服務大平台
由臺北市資訊局架設、維護，為整合臺北市政府各機關之申辦案件服務的平台，提供超過 1400 項線上申辦服務。

團隊經過前期研究後，發現「場地租借服務」是此次改造的潛在標的，除了因場地租用的服務四散在不同的營運方外，服務流程本身也相當繁複。因此，在設計方面，期能透過「效率查找」、「良好的資訊呈現」、「優化租借服務流程」三大目標，解決市民在租借場地上的痛點。
